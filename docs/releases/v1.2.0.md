# Bonjou v1.2.0

Bonjou v1.2.0 introduces metadata-first approval for transfers, stronger queue handling, and improved transfer status flows.

## Highlights

- Metadata-first approval queue for all incoming files and folders.
- New queue commands: `@queue`, `@view <id>`, `@approve <id>`, `@reject <id>`, `@approveAll`, `@rejectAll`.
- Pending approval queue is persisted and survives restarts.
- Clearer transfer lifecycle updates across sender and receiver.
