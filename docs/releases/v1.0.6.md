# Bonjou v1.0.6 Release Notes

## Highlights

- `@setname` now enforces clean handles (spaces collapse to dashes), persists the value, and triggers an immediate re-announce to peers.
- Discovery broadcasts across every active network interface; transfers inherit the refreshed identity data to eliminate stale usernames and IPs.
- Progress output remains on a single live line with readline fallbacks, and Windows consoles now enable ANSI sequences automatically.
- `@multi` accepts comma-separated handles even with stray spaces, easing simultaneous transfers to multiple peers.
- A comprehensive installation guide (`docs/install-guide.md`) now includes troubleshooting steps such as clearing the Scoop cache and running directly from source.

## Upgrade Instructions

- **Homebrew**
  ```bash
  brew update
  brew upgrade bonjou
  ```
- **Scoop**
  ```powershell
  scoop update
  scoop update bonjou
  ```
- **Manual**
  Download the appropriate artifact from the v1.0.6 release, place it on your PATH, and relaunch `bonjou`.

Encourage every teammate to refresh so usernames remain in sync and transfer output stays tidy across the LAN.
