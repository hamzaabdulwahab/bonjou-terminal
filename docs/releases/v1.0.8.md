# Bonjou v1.0.8 Release Notes

## Highlights

- Progress HUD clamps to the active terminal width using runewidth-aware truncation, keeping long filenames on a single line.
- Background IP watcher detects interface changes and refreshes peer discovery automatically after Wi-Fi or adapter switches.
- Completed transfers now leave behind a final 100% summary line so history stays readable after the bar clears.
- Windows terminals inherit ANSI colour support consistently, improving parity with macOS and Linux sessions.
- Packaging manifests refreshed for Scoop and Homebrew alongside updated install docs for the latest workflow.

## Upgrade Instructions

- **Homebrew**
  ```bash
  brew update
  brew upgrade bonjou
  ```
- **Scoop**
  ```powershell
  scoop update
  scoop update bonjou
  ```
- **Manual**
  Download the appropriate artifact from the v1.0.8 release, place it on your PATH, and relaunch `bonjou`.

Enjoy the smoother cross-network experience and tidier progress output!
