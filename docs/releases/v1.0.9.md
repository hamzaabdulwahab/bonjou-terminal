# Bonjou v1.0.9 Release Notes

## Highlights

- Windows consoles now render colour accents and borders identically to macOS/Linux, thanks to revised ANSI detection and fallbacks.
- The `@users` roster shows clean, port-free host names with friendly "last seen" phrasing so busy rooms stay readable.
- Message history gains a multi-line table that wraps long filenames and status updates without breaking alignment.
- Transfer notifications distinguish between files and folders, making mixed payloads easier to scan in real time.

## Upgrade Instructions

- **Homebrew**
  ```bash
  brew update
  brew upgrade bonjou
  ```
- **Scoop**
  ```powershell
  scoop update
  scoop update bonjou
  ```
- **Manual**
  Download the appropriate artifact from the v1.0.9 release, place it on your PATH, and relaunch `bonjou`.

Enjoy the sharper UI polish and the tidier history windows during your next LAN session!
