# Bonjou v1.0.0

Updated v1.0.0 release with security and reliability improvements.

## Highlights

- Enforced strict public-key-only transport key agreement.
- Removed secret-based legacy fallback for message/file transfer.
- Discovery now advertises `public_key` instead of sharing plaintext secret material.
- Added explicit transfer cancellation/failure status events.
- Improved username collision handling by auto-resolving duplicate names deterministically.

## Security Changes

- Transfer key derivation now requires discovered peer `public_key`.
- Peers without `public_key` are rejected for transfer/authentication.
- Envelope signature verification remains mandatory before payload processing.

## Behavior Changes

- Older nodes that only support legacy secret-based keying are no longer compatible.
- `@setname` now auto-adjusts duplicate names instead of failing outright.
- Mid-transfer shutdown/error now emits explicit cancellation/failure UI messages.

## Install

See [install guide](../install-guide.md) for full instructions.

**Mac:**
```bash
brew tap hamzaabdulwahab/bonjou https://github.com/hamzaabdulwahab/homebrew-bonjou
brew install bonjou
```

**Windows:**
```powershell
scoop bucket add bonjou https://github.com/hamzaabdulwahab/scoop-bonjou
scoop install bonjou
```

**Linux (Intel/AMD):**
```bash
wget https://github.com/hamzaabdulwahab/bonjou-cli/releases/download/v1.0.0/bonjou_1.0.0_amd64.deb
sudo dpkg -i bonjou_1.0.0_amd64.deb
```

**Linux (ARM64 - Docker/Parallels on Mac):**
```bash
wget https://github.com/hamzaabdulwahab/bonjou-cli/releases/download/v1.0.0/bonjou_1.0.0_arm64.deb
sudo dpkg -i bonjou_1.0.0_arm64.deb
```
