# Bonjou v1.0.5 Release Notes

## Highlights

- Streamlined transfer progress stays on a single line with direction icons, richer metadata, and calmer refresh cadence so large sends no longer flood the prompt.
- File and folder delivery now respects configurable `chunk_size_bytes` (4 KiB – 4 MiB) and `chunk_timeout_secs` (5 – 600 s) in `~/.bonjou/config.json`, helping tune throughput for noisy LANs.
- `@help` is reorganized into color-coded sections covering messaging, transfer, discovery, and maintenance commands for faster lookup during a session.

## Upgrade Instructions

- **Homebrew**
  ```bash
  brew update
  brew upgrade bonjou
  ```
- **Scoop**
  ```powershell
  scoop update
  scoop update bonjou
  ```
- **Manual**
  Download the appropriate artifact from the v1.0.5 release, place it on your PATH, and relaunch `bonjou`.

Encourage every teammate to refresh so buttery progress output and the new transfer tuning options are consistent across the LAN.
