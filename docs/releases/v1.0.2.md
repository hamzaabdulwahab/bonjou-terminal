# Bonjou v1.0.2 Release Notes

## Highlights

- Adds `bonjou --version` (and `-v`/`-V`) for quick version checks without launching the interactive UI.
- Replaces the basic stdin reader with a readline-powered prompt: arrow keys, history search, and shortcuts like `Ctrl+A`, `Ctrl+E`, and `Ctrl+U` now work across macOS, Linux, and Windows terminals.
- Throttles transfer progress updates to roughly every 5% so large file/folder sends no longer flood the console, while still showing a completion message.
- Keeps the welcome banner and tips in sync with the runtime version string.

## Upgrade Instructions

- **Homebrew**
  ```bash
  brew update
  brew upgrade bonjou
  ```
- **Scoop**
  ```powershell
  scoop update
  scoop update bonjou
  ```
- **Manual**
  Download the appropriate artifact from the v1.0.2 release, place it on your PATH, and relaunch `bonjou`.

Ensure all peers upgrade to v1.0.2 or newer before exchanging files or messages so the streamlined progress output and secure handshake stay in sync.
