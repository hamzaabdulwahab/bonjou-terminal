# Bonjou v1.0.4 Release Notes

## Highlights

- Tightens CLI usage so only plain `bonjou` launches the UI; `--version` still reports the build, but any other flags are rejected with a helpful message.
- Introduces a centered ASCII welcome banner inspired by binsider, complete with a tagline and GitHub link, while keeping the session details visible below.

## Upgrade Instructions

- **Homebrew**
  ```bash
  brew update
  brew upgrade bonjou
  ```
- **Scoop**
  ```powershell
  scoop update
  scoop update bonjou
  ```
- **Manual**
  Download the appropriate artifact from the v1.0.4 release, place it on your PATH, and relaunch `bonjou`.

Encourage every teammate to upgrade so the consistent startup experience and stricter CLI behavior apply across the LAN.
