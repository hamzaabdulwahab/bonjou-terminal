# Bonjou v1.1.0

This refreshed v1.1.0 release keeps the same version while updating binaries and packaging metadata.

## Highlights

- Added a polished custom wizard theme (purple/blue accents, rounded focus cards, clearer button states).
- Improved interactive command flow by applying consistent styling to all prompt forms.
- Reworked many network/crypto transfer errors into user-friendly messages with clear next steps.
- Strengthened config safety by failing fast if secure random generation is unavailable.

## Notes

- This is a rewrite of the existing `v1.1.0` release, not a new version bump.
- Package manager definitions (Homebrew, Scoop, WinGet manifest source) were updated with fresh checksums.
