# Bonjou v1.1.0

This release introduces full transport privacy and stronger delivery confirmation reliability.

## Highlights

- Full encryption for everything on the transfer channel.
- Metadata/header fields are now encrypted, not only payload bytes.
- Delivery acknowledgements improved for repeated transfers and rapid re-sends.
- UI now clearly separates `sent` from `delivered` states.

## Security Update (Must Read)

In `v1.1.0`, Bonjou encrypts the complete transfer envelope and payload.
This includes transfer metadata such as sender/receiver identifiers, names, and transfer attributes.

## Reliability Update

- Added inline delivery ACK handling on the same transfer connection.
- Added ACK fallback handling to reduce intermittent timeout/EOF confirmation issues.
- Reduced event-drop pressure so receive/send status messages are more consistent.
