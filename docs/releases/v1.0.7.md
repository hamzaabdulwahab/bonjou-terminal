# Bonjou v1.0.7 Release Notes

## Highlights

- Discovery now enables UDP broadcast sockets using platform-specific code paths, fixing Windows cross-compilation and ensuring peers are found across every active NIC.
- Username changes via `@setname` normalise spaces to dashes, persist immediately, and re-announce to the LAN without waiting for the next tick.
- `@multi` accepts comma- or space-separated handles interchangeably, simplifying multi-recipient transfers while preserving sequential delivery.
- Windows consoles automatically opt into ANSI mode, keeping colours and inline progress tidy; limited terminals fall back gracefully.
- Installation docs gained Scoop cache troubleshooting, checksum refreshes, and a quick "go run" workflow for local testing.

## Upgrade Instructions

- **Homebrew**
  ```bash
  brew update
  brew upgrade bonjou
  ```
- **Scoop**
  ```powershell
  scoop update
  scoop update bonjou
  ```
- **Manual**
  Download the appropriate artifact from the v1.0.7 release, place it on your PATH, and relaunch `bonjou`.

Encourage teammates to update so everyone benefits from the discovery fixes, ANSI polish, and stricter username handling.
