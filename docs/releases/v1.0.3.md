# Bonjou v1.0.3 Release Notes

## Highlights

- Normalizes `@setpath` arguments so `~/Downloads` and relative paths resolve to the correct absolute location before being saved.
- Restores LAN-wide messaging: `@broadcast` now reuses the discovery secret for each peer so messages arrive without "peer secret unknown" failures.
- Tames progress noise by keeping 5% increments, skipping redundant 100% lines, and spacing updates by ~750 ms for large transfers.

## Upgrade Instructions

- **Homebrew**
  ```bash
  brew update
  brew upgrade bonjou
  ```
- **Scoop**
  ```powershell
  scoop update
  scoop update bonjou
  ```
- **Manual**
  Download the appropriate artifact from the v1.0.3 release, place it on your PATH, and relaunch `bonjou`.

Encourage the whole team to update so the quieter progress output and broadcast reliability stay consistent across the LAN.
